# "水"特征和页面显示问题修复完成

## 修复时间
2026-03-05

## 问题描述

### 问题1：设备管理页面描述文字显示不完整
用户报告：设备管理页面顶部的描述文字"管理设备库中的设备信息，支持添加、编辑、删除和批量导入"在设备数量较多时显示不完整。

### 问题2："水"特征未被添加到规则中
用户报告：虽然在设备详情中可以看到"介质: 水"，但在匹配规则中找不到"水"这个特征。

## 根本原因分析

### 问题1原因
`DeviceManagementView.vue` 中的 `.header-card p` 样式缺少文本换行相关的CSS属性，导致长文本被截断。

### 问题2原因
通过诊断脚本 `diagnose_water_feature.py` 发现了两个问题：

1. **配置问题**：虽然已经添加了 `medium_keywords` 字段，但"介质"这个元数据标签本身不在 `metadata_keywords` 列表中
2. **元数据标签处理问题**：当设备参数以"介质: 水"格式存储时，预处理器无法正确删除"介质:"前缀，导致提取的特征是"介质:水"而不是"水"

## 修复方案

### 修复1：设备管理页面描述显示
**文件**: `frontend/src/views/DeviceManagementView.vue`

**修改内容**:
```css
.header-card p {
  margin: 0;
  color: #909399;
  font-size: 14px;
  white-space: normal;      /* 新增：允许文本换行 */
  word-wrap: break-word;    /* 新增：长单词自动换行 */
  line-height: 1.5;         /* 新增：增加行高，提高可读性 */
}
```

**效果**:
- 描述文字现在可以正常换行显示
- 即使设备列表很长，描述文字也能完整显示
- 行高增加，提高了可读性

### 修复2："水"特征提取
**文件**: `data/static_config.json`

**修改内容**:
在 `metadata_keywords` 数组中添加"介质"：
```json
"metadata_keywords": [
  "型号",
  "通径",
  "阀体类型",
  "适用介质",
  "介质",        // 新增
  "品牌",
  ...
]
```

**原理**:
- `metadata_keywords` 用于标识哪些词是字段名称（元数据标签），不应该作为特征
- 当文本预处理器遇到"介质: 水"时，会识别"介质"为元数据标签
- 删除"介质:"前缀后，只保留"水"作为特征
- "水"在 `medium_keywords` 中，质量评分为65分（基础50分 + 在设备关键词库中15分）
- 65分 > 最小阈值50分，因此"水"会被保留

## 技术细节

### 特征质量评分机制
"水"的质量评分计算：
- 基础分：50
- 在设备关键词库中（medium_keywords）：+15
- 总分：65

评分规则（来自 `text_preprocessor.py`）：
```python
def _in_device_keywords(self, feature: str) -> bool:
    """判断是否在设备关键词库中"""
    # 检查品牌关键词
    for brand in self.brand_keywords:
        if brand.lower() in feature.lower():
            return True
    
    # 检查设备类型关键词
    for device_type in self.device_type_keywords:
        if device_type in feature:
            return True
    
    # 检查介质关键词（新增）
    for medium in self.medium_keywords:
        if medium in feature:
            return True
    
    return False
```

### 元数据标签删除机制
预处理器的 `remove_metadata_labels` 方法会：
1. 按关键词长度从长到短排序（避免误匹配）
2. 对每个关键词生成正则表达式：`(?:\d+\.?)?{keyword}[:：]`
3. 匹配并删除元数据标签，保留值部分

示例：
- "介质: 水" → 删除"介质:" → "水"
- "2.型号:QAA2061" → 删除"2.型号:" → "QAA2061"
- "适用介质: 水" → 删除"适用介质:" → "水"
