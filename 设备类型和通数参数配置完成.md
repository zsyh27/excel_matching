## 示例：霍尼韦尔座阀录入方式

### 原始数据
```
霍尼韦尔 座阀 二通+DN80+蒸汽+V5211F2010+ML8824A1820+V5211+ML8824系列
型号：V5211F2010+ML8824A1820
阀体类型：二通阀
通径：3"(DN80)
适用介质：蒸汽
配套执行器：ML8824A1820
```

### 推荐录入方式

**基本信息**:
- 品牌: `霍尼韦尔`
- 设备类型: `电动调节阀`
- 设备名称: `座阀`
- 规格型号: `V5211F2010+ML8824A1820`
- 单价: (根据实际填写)

**关键参数** (key_params):
- 通径: `DN80` 或 `3"(DN80)` (必填)
- 通数: `二通`
- 阀体类型: `座阀`
- 执行器品牌: `霍尼韦尔`
- 执行器型号: `ML8824A1820`
- 介质: `蒸汽`

**详细参数** (detailed_params，可选):
- 可以留空，或填写其他补充信息

### 权重分配说明

根据当前配置，特征权重分配如下:
- device_type (电动调节阀): 20.0 (最高)
- key_params (通径、通数、阀体类型等): 15.0 (次高)
- brand (霍尼韦尔): 10.0 (中高)
- model (V5211F2010+ML8824A1820): 5.0 (中)
- parameter (详细参数): 1.0 (低)

**关键优势**:
- "通数"作为key_params，权重为15.0，远高于detailed_params的1.0
- 二通阀和三通阀可以有效区分，提高匹配准确度
