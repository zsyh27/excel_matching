# 设备参数配置功能实现说明

## 功能概述

设备参数配置功能允许用户可视化地配置每种设备类型对应的参数字段，用于动态表单生成和Excel模板生成。

## 实现内容

### 1. 前端组件集成

#### 1.1 创建 DeviceParamsEditor 组件
- 位置：`frontend/src/components/ConfigManagement/DeviceParamsEditor.vue`
- 功能：
  - 左侧面板：显示设备类型列表
  - 右侧面板：编辑选中设备类型的参数
  - 支持添加/删除设备类型
  - 支持添加/删除参数
  - 参数字段：
    - 参数名称（如：量程、输出信号、精度）
    - 是否必填
    - 数据类型（字符串、数字、范围）
    - 单位（如：ppm、mA、℃）
    - 提示信息

#### 1.2 集成到配置管理页面
- 在 `ConfigManagementView.vue` 中添加菜单项
- 菜单位置：匹配配置阶段，位于"设备类型"和"特征权重"之间
- 菜单图标：📋
- 菜单标签：设备参数配置

### 2. 后端配置数据

#### 2.1 配置文件更新
- 文件：`data/static_config.json`
- 新增字段：`device_params`
- 数据结构：
```json
{
  "device_params": {
    "设备类型名称": {
      "params": [
        {
          "name": "参数名称",
          "required": true/false,
          "data_type": "string/number/range",
          "unit": "单位",
          "hint": "提示信息"
        }
      ]
    }
  }
}
```

#### 2.2 初始配置
已为以下设备类型添加了初始参数配置：

1. **CO2传感器**
   - 量程（必填，范围，ppm）
   - 输出信号（必填，字符串，mA）
   - 精度（可选，字符串）

2. **温度传感器**
   - 量程（必填，范围，℃）
   - 输出信号（必填，字符串）
   - 精度（可选，字符串）

3. **座阀**
   - 通径（必填，数字，mm）
   - 压力等级（可选，数字，bar）
   - 流量系数（可选，数字）

4. **执行器**
   - 扭矩（可选，数字，N·m）
   - 行程时间（可选，数字，s）
   - 控制信号（可选，字符串）

## 使用方法

### 访问设备参数配置界面

1. 打开系统，进入"配置管理"页面
2. 在左侧菜单中找到"设备参数配置"（📋图标）
3. 点击进入设备参数配置界面

### 编辑设备参数

1. **选择设备类型**
   - 在左侧面板中点击要编辑的设备类型
   - 右侧将显示该设备类型的参数列表

2. **添加参数**
   - 点击右上角"添加参数"按钮
   - 填写参数信息：
     - 参数名称：如"量程"、"输出信号"
     - 是否必填：开关切换
     - 数据类型：选择字符串、数字或范围
     - 单位：如"ppm"、"mA"、"℃"
     - 提示信息：为用户提供填写示例

3. **删除参数**
   - 点击参数右上角的删除按钮

4. **添加设备类型**
   - 点击左上角"添加类型"按钮
   - 输入设备类型名称
   - 点击确定

5. **删除设备类型**
   - 选中要删除的设备类型
   - 点击右上角"删除类型"按钮
   - 确认删除

6. **保存配置**
   - 编辑完成后，点击页面顶部的"保存"按钮
   - 输入备注信息（可选）
   - 确认保存

## 设计说明

### 为什么不包含正则表达式？

根据用户需求，当前版本的设备参数配置**不包含正则表达式编辑功能**。原因如下：

1. **简化用户体验**：正则表达式对非技术用户来说过于复杂
2. **避免Excel文本解析**：暂时不考虑从Excel文本中自动提取参数
3. **结构化数据输入**：未来计划采用结构化的Excel模板，直接通过列来输入参数值

### 未来规划

1. **Excel模板生成**
   - 根据设备参数配置动态生成Excel模板
   - 模板包含设备类型下拉框
   - 为每个参数提供独立的列（Param1, Param2, Param3等）
   - 包含"参数说明"工作表，映射设备类型到参数含义

2. **批量导入优化**
   - 修改批量导入逻辑，直接读取结构化参数列
   - 不再依赖文本解析和正则表达式
   - 提高数据准确性和导入效率

3. **动态表单生成**
   - 在设备录入表单中，根据选择的设备类型动态显示参数字段
   - 自动应用必填验证和数据类型验证
   - 显示单位和提示信息

## 技术实现

### 前端技术栈
- Vue 3 Composition API
- Element Plus UI组件库
- 响应式数据绑定

### 数据流
1. 用户编辑 → 组件本地状态更新
2. 点击保存 → 调用配置API
3. 后端保存 → 更新 static_config.json
4. 返回成功 → 刷新配置缓存

### 配置持久化
- 配置保存到 `data/static_config.json`
- 支持版本历史记录
- 支持配置回滚
