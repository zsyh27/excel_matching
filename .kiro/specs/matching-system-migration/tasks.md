# 匹配系统迁移 - 任务清单

## 任务概述

本任务清单用于将现有的智能特征提取系统完全替换旧的匹配引擎，实现统一的五步流程处理。

**重要说明**：
- ✅ 智能特征提取系统的核心功能已在 `.kiro/specs/intelligent-feature-extraction` 中实现
- ✅ 五步流程API已实现：`/api/intelligent-extraction/match`、`/api/intelligent-extraction/preview`
- ✅ 文本预处理（归一化、同义词映射）已在 `TextPreprocessor` 中实现
- ✅ 本任务清单专注于**系统迁移和集成工作**，避免重复开发
- ✅ 核心工作已完成：旧的 `/api/match` 已迁移到使用新的 `IntelligentExtractionAPI`

**五步流程**：设备类型识别 → 参数提取 → 辅助信息提取 → 智能匹配 → UI展示

**关键说明**：
- ✅ 删除步骤0（文本预处理）- 归一化/同义词映射在步骤1-3提取后应用
- ✅ 测试页面已从六步改为五步流程显示
