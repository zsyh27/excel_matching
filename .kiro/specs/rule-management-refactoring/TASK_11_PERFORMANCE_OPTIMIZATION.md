## 缓存管理

### 缓存失效策略
1. **自动失效**
   - 设备创建/更新/删除时失效设备缓存
   - 规则创建/更新/删除时失效规则缓存
   - 匹配操作时失效统计缓存

2. **定时清理**
   - 每5分钟自动清理过期缓存
   - 避免内存泄漏

3. **手动刷新**
   - 前端提供刷新按钮
   - 用户可以手动刷新数据

### 缓存键设计
```
设备列表: devices:page={page}:size={size}:name={name}:brand={brand}:has_rule={has_rule}
设备详情: device:{device_id}
规则统计: statistics:rules
匹配日志: statistics:logs:page={page}:status={status}:date={date}
成功率: statistics:success-rate:start={start}:end={end}
```
