## 核心功能

### 1. 设备类型识别

**识别模式**：
- 精确匹配（置信度100%）：CO浓度探测器 → CO浓度探测器
- 模糊匹配（置信度90%）：CO探测器 → CO浓度探测器
- 关键词匹配（置信度80%）：CO + 探测器 → CO浓度探测器
- 类型推断（置信度70%）：CO → CO浓度探测器

**测试覆盖**：
- ✅ 属性测试：总是返回置信度
- ✅ 属性测试：识别准确率 >85%
- ✅ 属性测试：参数不匹配时设备类型仍然正确
- ✅ 单元测试：精确匹配、模糊匹配、关键词匹配、空输入

### 2. 参数提取

**支持的参数类型**：
- 量程：0~250ppm, -40~80℃
- 输出信号：4~20mA, 0~10V, RS485
- 精度：±5%, ±0.5℃
- 规格：DN50, PN16, PT100

**归一化处理**：
- 量程：{min: 0, max: 250, unit: "ppm"}
- 输出：{min: 4, max: 20, unit: "mA", type: "analog"}
- 精度：{value: 5, unit: "%"}

**测试覆盖**：
- ✅ 属性测试：参数归一化的一致性
- ✅ 属性测试：参数提取总是返回置信度
- ✅ 单元测试：带标签提取、无标签提取、各种参数类型

### 3. 智能匹配

**评分算法**：
- 设备类型得分：50%
- 参数得分：30%
- 品牌得分：10%
- 其他得分：10%

**匹配策略**：
- 严格匹配（90+分）
- 宽松匹配（70-89分）
- 模糊匹配（50-69分）
- 兜底匹配（30-49分）

**测试覆盖**：
- ✅ 属性测试：评分权重的正确性
- ✅ 属性测试：候选设备按评分降序排列
- ✅ 单元测试：评分计算、排序功能

### 4. API接口

**提供的接口**：
- `extract(text)`: 提取设备信息
- `match(text, top_k)`: 智能匹配设备
- `match_batch(items, top_k)`: 批量匹配
- `preview(text)`: 五步流程预览

**性能指标**：
- 响应时间：<500ms
- 批量处理速度：>100条/秒

## 使用示例

### 基本使用

```python
from modules.intelligent_extraction.api_handler import IntelligentExtractionAPI

# 初始化
api = IntelligentExtractionAPI(config, device_loader)

# 提取设备信息
result = api.extract("CO浓度探测器 量程0~250ppm 输出4~20mA")

# 智能匹配
result = api.match("CO浓度探测器 量程0~250ppm 输出4~20mA", top_k=5)

# 五步流程预览
result = api.preview("CO浓度探测器 量程0~250ppm 输出4~20mA")
```
