# 智能特征提取系统 - 用户验收测试计划

## 测试概述

**测试目标**: 验证智能特征提取系统满足用户需求,功能正常,性能达标

**测试时间**: 2026-03-07 - 2026-03-08

**测试人员**: 最终用户 + 开发团队

## 测试环境

### 硬件环境
- 操作系统: Windows 10/11
- 浏览器: Chrome 最新版
- 内存: ≥8GB
- 网络: 局域网

### 软件环境
- 后端: Python 3.8+, Flask
- 前端: Vue.js 3, Element Plus
- 数据库: SQLite 3 (171个设备)

### 测试数据
- 现有设备库: 171个设备
- 测试设备描述: 30个样本
- Excel测试文件: 包含10个设备

## 测试用例

### TC01: 单个设备智能识别

**测试目标**: 验证单个设备的智能识别功能

**前置条件**: 
- 系统已启动
- 用户已登录
- 进入设备录入界面

**测试步骤**:
1. 在设备名称输入框输入"室内温度传感器"
2. 点击"智能识别"按钮
3. 查看识别结果

**预期结果**:
- ✅ 识别出设备类型: "温度传感器"
- ✅ 置信度: ≥90%
- ✅ 响应时间: <1秒
- ✅ 显示识别结果预览

**测试数据**:
```
输入1: 室内温度传感器
输入2: 风管温湿度传感器
输入3: 室内空气质量传感器
输入4: CO浓度探测器 量程0~250ppm 输出4~20mA
输入5: 温湿度传感器 霍尼韦尔 HST-RA
```

**验收标准**:
- [ ] 5个测试用例全部识别正确
- [ ] 平均置信度 ≥85%
- [ ] 平均响应时间 <1秒

## 问题分类和处理

### 严重问题 (P0)
- 系统崩溃
- 数据丢失
- 功能完全不可用
- **处理**: 立即修复,阻塞上线

### 重要问题 (P1)
- 识别准确率不达标
- 性能不达标
- 用户体验差
- **处理**: 优先修复,影响上线

### 一般问题 (P2)
- 界面细节问题
- 提示信息不清晰
- 小功能缺失
- **处理**: 记录跟踪,后续优化
