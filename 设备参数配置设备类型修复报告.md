## 用户操作指南

### 访问设备参数配置
1. 打开浏览器访问：http://localhost:5173
2. 进入"配置管理"页面
3. 点击左侧菜单"设备参数配置"

### 预期效果
- 左侧面板显示36个设备类型
- 每个设备类型显示参数数量（3个参数）
- 点击任意设备类型，右侧显示详细参数配置
- 可以添加、编辑、删除参数

### 功能说明
- **添加类型**：点击"添加类型"按钮可以新增设备类型
- **选择类型**：点击左侧设备类型可以查看和编辑参数
- **添加参数**：在右侧点击"添加参数"可以为当前设备类型添加新参数
- **删除类型**：选中设备类型后，点击"删除类型"可以移除该类型
- **参数配置**：每个参数包含：
  - 参数名称（如：量程、通径）
  - 是否必填（开关）
  - 数据类型（字符串/数字/范围）
  - 单位（如：ppm、mm、℃）
  - 提示信息（帮助用户填写）

## 技术细节

### 脚本功能
`add_all_device_params.py` 脚本的主要功能：
1. 连接数据库 `data/devices.db`
2. 将36种设备类型的参数配置转换为JSON
3. 更新 `configs` 表中的 `device_params` 记录
4. 验证更新结果并显示统计信息

### 数据结构
```json
{
  "设备类型名称": {
    "params": [
      {
        "name": "参数名称",
        "required": true/false,
        "data_type": "string/number/range",
        "unit": "单位",
        "hint": "提示信息"
      }
    ]
  }
}
```
