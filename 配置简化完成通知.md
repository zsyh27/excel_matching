## 🚀 如何使用

### 访问配置管理页面
1. 启动后端：`cd backend && python app.py`
2. 访问前端：`http://localhost:5000`
3. 点击"配置管理"菜单

### 配置全局选项
1. 进入"全局配置"
2. 启用智能拆分（如需要）
3. 添加元数据标签（如需要）
4. 调整匹配阈值（推荐5.0）

### 配置同义词映射
1. 进入"智能提取配置" → "同义词映射"
2. 选择类型（同义词/技术术语/单位归一化）
3. 添加映射规则
4. 保存配置
