# 匹配增强功能使用指南

## 新功能概述

我们为匹配系统添加了三个增强功能，帮助您提高匹配准确度：

### 1. 智能拆分 🔀
自动拆分复合词和技术规格，提高匹配灵活性。

**示例**：
- "室内墙装" → 可以匹配"室内吊装"
- "ntc 10k" → 可以匹配"NTC10K"
- "DN15" → 可以匹配"通径15"

**配置位置**：配置管理 → 处理分隔符（第6项）

### 2. 单位删除 📏
自动删除单位符号，避免因单位不同导致匹配失败。

**示例**：
- "0-2000ppm" → "0-2000"
- "4-20ma" → "4-20"
- "-20~60℃" → "-20~60"

**配置位置**：配置管理 → 高级配置（第8项）

### 3. 技术规格拆分 🔧
将复杂参数拆分为简单特征，提高匹配召回率。

**示例**：
- "ntc 10k" → ["ntc", "10k"]
- "-20~60℃" → ["-20", "60"]

**配置位置**：配置管理 → 智能清理（第5项）

## 快速开始

### 推荐配置（适合大多数场景）

1. **启用智能拆分**：
   - 打开"配置管理" → "处理分隔符"
   - 勾选"启用智能拆分"
   - 勾选所有拆分选项
   - 点击"保存配置"

2. **配置单位删除**：
   - 打开"配置管理" → "高级配置"
   - 勾选"启用单位删除"
   - 添加常用单位：ppm、ma、v、pa、℃、c、%rh
   - 点击"保存配置"

3. **测试效果**：
   - 进行一次匹配测试
   - 查看匹配详情，确认效果
   - 根据需要调整配置

## 重要说明

### ✅ 数据安全
- 这些功能**只在匹配阶段生效**
- **设备录入数据保持完整**，不会被修改
- 您可以随时启用或禁用这些功能

### 📊 效果预期
- **提高召回率**：能匹配到更多相似设备
- **保持准确度**：不会降低匹配质量
- **灵活配置**：根据实际需求调整

### ⚠️ 注意事项
1. 首次使用建议先测试，确认效果符合预期
2. 如果匹配结果不理想，可以调整配置或禁用某些功能
3. 单位列表需要根据实际业务场景配置

## 常见问题

**Q: 这些功能会影响已有的设备数据吗？**
A: 不会。这些功能只在匹配阶段生效，不会修改设备库中的数据。

**Q: 如果效果不好，可以关闭吗？**
A: 可以。所有功能都可以随时启用或禁用，不会影响系统稳定性。

**Q: 需要重新导入设备吗？**
A: 不需要。这些功能对现有设备数据自动生效。

**Q: 如何知道哪些特征被拆分了？**
A: 在匹配详情页面可以查看完整的特征提取过程，包括拆分详情。
