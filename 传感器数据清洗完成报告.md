## 📝 技术说明

### 数据清洗逻辑

#### 前3组（室内/风管/室外传感器）
- 根据型号前缀识别设备类型（HST-=温度，HSH-=温湿度）
- 直接映射字段到key_params
- 添加默认量程信息

#### 第4组（电流电压输出型）
- 从"产品类型"字段提取设备类型和安装位置
- 解析输出信号、传感器类型、精度等参数
- 添加默认量程信息

#### 第5组（现场设备）
使用智能解析算法：
1. **设备类型识别**: 关键词匹配（空气质量、CO2、CO、PM等）
2. **检测对象提取**: 识别温度、湿度、CO2、PM2.5等
3. **量程解析**: 正则表达式提取（如"0-100PPM"）
4. **输出信号提取**: 识别4-20mA、0-10V、Modbus等
5. **其他特征**: 显示屏、继电器、颜色、电源等

### key_params结构设计

每个参数包含：
- `value`: 参数值
- `unit`: 单位（可选）
- `required`: 是否必填

示例：
```json
{
  "量程": {
    "value": "0-100",
    "unit": "ppm",
    "required": true
  }
}
```
