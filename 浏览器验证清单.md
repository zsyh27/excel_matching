## 常见问题

### Q1: 步骤2仍然显示旧格式（3个参数）

**原因**: 前端代码未更新或浏览器缓存

**解决方案**:
1. 确认 `frontend/src/components/Testing/FiveStepPreview.vue` 已更新
2. 重启前端开发服务器（如果使用）
3. 强制刷新浏览器（Ctrl+Shift+R）
4. 清除浏览器缓存

### Q2: 设备参数列表没有高亮

**原因**: 后端代码未更新或Python缓存未清除

**解决方案**:
1. 确认 `backend/modules/intelligent_extraction/intelligent_matcher.py` 已更新
2. 删除所有 `__pycache__` 文件夹
3. 重启后端服务
4. 刷新浏览器

### Q3: 未匹配参数仍显示"全部匹配"

**原因**: 后端代码未更新

**解决方案**:
1. 确认 `_score_device()` 方法中的 `unmatched_params` 计算逻辑已更新
2. 重启后端服务
3. 刷新浏览器

### Q4: API返回错误

**检查步骤**:
1. 打开浏览器开发者工具（F12）
2. 切换到 Network 标签
3. 查看 `/api/intelligent-extraction/preview` 请求
4. 检查响应状态码和错误信息
5. 查看后端控制台日志
