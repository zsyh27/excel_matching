## 📊 数据格式要求

### 最终Excel格式（系统要求）

Excel文件必须包含以下列：

| 列名 | 必填 | 说明 | 示例 |
|------|------|------|------|
| 品牌 | ✅ | 设备品牌 | 霍尼韦尔 |
| 设备类型 | ✅ | 设备类型（权重20.0） | 压差变送器 |
| 设备名称 | ✅ | 设备名称 | 陶瓷芯体压差变送器 |
| 规格型号 | ✅ | 型号 | P7620C0042A |
| 单价 | ✅ | 价格（整数） | 4516 |
| **参数列** | ⚠️ | 各种参数（自动转为key_params） | 量程、输出信号、精度等 |

**重要说明**：
- 前5列是标准字段（必填）
- 第6列及之后的所有列会自动识别为 `key_params`（权重15.0）
- 参数列名称可以自定义（如：量程、输出信号、精度、通径等）
- 参数列越详细，匹配准确度越高

## 💡 提供数据的最佳方式

### 方式1：纯文本格式（推荐）⭐

**优点**：最简单，直接复制粘贴

**格式**：每行一个设备，字段用空格或制表符分隔

```
序号 型号 价格 参数描述
82 P7620C0042A 4516 0...4 Bar, H-Port<20 Bar, L-Port<20 Bar, TC-0<0.08%, 4-20mA
83 P7620C0042B 4516 0...4 Bar, H-Port<20 Bar, L-Port<20 Bar, TC-0<0.08%, 0-10V
```

**提供方式**：
1. 保存为 `.txt` 文件
2. 或直接在聊天中粘贴

### 方式2：结构化文本（推荐用于复杂数据）⭐⭐

**优点**：字段清晰，易于解析

**格式**：使用明确的字段分隔符

```
型号: P7620C0042A | 价格: 4516 | 量程: 0-4 Bar | 输出: 4-20mA | 精度: TC-0<0.08%
型号: P7620C0042B | 价格: 4516 | 量程: 0-4 Bar | 输出: 0-10V | 精度: TC-0<0.08%
```

### 方式3：CSV格式

**优点**：适合大量数据

**格式**：逗号分隔值

```csv
型号,价格,量程,输出信号,精度
P7620C0042A,4516,0-4 Bar,4-20mA,TC-0<0.08%
P7620C0042B,4516,0-4 Bar,0-10V,TC-0<0.08%
```

### 方式4：Excel文件

**优点**：保留原始格式

**提供方式**：
1. 上传Excel文件
2. 或复制Excel内容粘贴为文本

## 🔧 Excel生成流程

### 我会执行的步骤：

1. **数据解析**
   - 识别数据格式
   - 提取字段信息
   - 统计数据量

2. **设备类型识别**
   - 根据型号和描述识别设备类型
   - 分类统计

3. **参数提取**
   - 从描述文本中提取结构化参数
   - 根据设备类型提取对应参数
   - 常见参数：量程、输出信号、精度、通径、介质等

4. **生成Excel**
   - 创建标准字段列
   - 创建参数列（自动转为key_params）
   - 设置样式和列宽
   - 保存到 `data/` 目录

5. **验证和统计**
   - 统计设备类型分布
   - 统计参数覆盖率
   - 输出生成报告

## 🎯 最佳实践

### ✅ 推荐做法

1. **提供完整数据**
   - 一次性提供所有数据（不要分批）
   - 包含所有字段信息

2. **说明数据格式**
   - 简单说明每列的含义
   - 指出哪些是重要参数

3. **指定品牌**
   - 明确告诉我品牌名称
   - 如果有多个品牌，分别说明

4. **提供示例**
   - 如果数据格式复杂，提供1-2条示例
   - 说明期望的输出格式

### ❌ 避免的做法

1. **不要分批提供**
   - 一次提供14条，后来说还有106条 ❌
   - 应该一次性提供全部120条 ✅

2. **不要省略字段说明**
   - 直接粘贴数据，不说明格式 ❌
   - 应该简单说明字段含义 ✅

3. **不要混合多种格式**
   - 同一批数据使用不同格式 ❌
   - 应该统一格式 ✅

## 🔍 常见设备类型和参数

### 压差变送器
- 量程（如：0-4 Bar）
- H-Port、L-Port
- 精度（如：TC-0<0.08%）
- 输出信号（4-20mA / 0-10V）
- 通讯方式（Modbus Rtu）
- 显示（带显示/无显示）

### 温度传感器
- 传感器类型（10K NTC / 20K NTC / PT1000）
- 长度（65mm / 102mm / 152mm / 300mm）
- 输出信号（4-20mA / 0-10V）

### 压力传感器
- 量程（如：10Bar / 16Bar / 20Bar）
- 精度（1% / 0.5% / 0.25%）
- 输出信号（4-20mA / 0-10V）
- 接口类型（G1/4 / G1/2 / NPT）
- 特性（高温型）

### 液位传感器
- 温度范围（如：-10 - +80°C）
- 输出信号（4-20mA）
- 量程（1m / 2m / 3m / 5m / 10m / 20m / 50m）

### 水流开关
- 压力等级（1MPa / 2MPa）

### 液位开关
- 线长（5米 / 10米 / 20米）
