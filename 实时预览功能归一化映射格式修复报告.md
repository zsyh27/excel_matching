# 实时预览功能归一化映射格式修复报告

## 问题描述

配置管理页面的实时预览功能报错：

```
ERROR:__main__:测试配置失败: 'list' object has no attribute 'items'
ERROR:__main__:Traceback (most recent call last):
  File "D:\excel_matching\excel_matching\backend\app.py", line 3678, in test_config
    preprocess_result = test_preprocessor.preprocess(test_text)
  File "D:\excel_matching\excel_matching\backend\modules\text_preprocessor.py", line 230, in preprocess
    normalized_text, normalization_detail = self._normalize_with_detail(cleaned_text, mode=mode)
  File "D:\excel_matching\excel_matching\backend\modules\text_preprocessor.py", line 352, in _normalize_with_detail
    sorted_mappings = sorted(self.normalization_map.items(), key=lambda x: len(x[0]), reverse=True)
AttributeError: 'list' object has no attribute 'items'
```

## 根本原因

### 数据格式不一致

1. **前端格式**（NormalizationEditor.vue）：
   ```javascript
   // 列表格式
   [
     { from: "温度", to: "温度" },
     { from: "温湿度", to: "温度" },
     { from: "测温", to: "温度" }
   ]
   ```

2. **后端期望格式**（text_preprocessor.py）：
   ```python
   # 字典格式
   {
     "温度": "温度",
     "温湿度": "温度",
     "测温": "温度"
   }
   ```

### 问题发生位置

在 `text_preprocessor.py` 的第 352 行：
```python
sorted_mappings = sorted(self.normalization_map.items(), key=lambda x: len(x[0]), reverse=True)
```

代码调用了 `.items()` 方法，这是字典的方法，但 `normalization_map` 实际上是列表。

## 修复方案

### 修改 text_preprocessor.py 初始化代码

在 `TextPreprocessor.__init__()` 方法中添加格式兼容处理：

```python
# 处理 normalization_map：支持列表和字典两种格式
normalization_map_raw = config.get('normalization_map', {})
if isinstance(normalization_map_raw, list):
    # 列表格式：[{"from": "旧值", "to": "新值"}, ...]
    self.normalization_map = {
        item['from']: item['to'] 
        for item in normalization_map_raw 
        if 'from' in item and 'to' in item
    }
else:
    # 字典格式：{"旧值": "新值", ...}
    self.normalization_map = normalization_map_raw
```

### 修复逻辑

1. **检测格式**：使用 `isinstance()` 判断是列表还是字典
2. **列表转字典**：如果是列表，遍历每个项目，提取 `from` 和 `to` 字段，构建字典
3. **保持兼容**：如果是字典，直接使用
4. **容错处理**：只处理包含 `from` 和 `to` 字段的项目

## 技术细节

### 格式转换逻辑

```python
# 输入（列表格式）
[
    {"from": "温湿度", "to": "温度"},
    {"from": "测温", "to": "温度"}
]

# 转换过程
{
    item['from']: item['to'] 
    for item in normalization_map_raw 
    if 'from' in item and 'to' in item
}

# 输出（字典格式）
{
    "温湿度": "温度",
    "测温": "温度"
}
```

### 容错处理

1. **缺少字段**：如果项目缺少 `from` 或 `to` 字段，跳过该项目
2. **空列表**：如果列表为空，返回空字典
3. **None 值**：如果配置为 None，使用默认空字典
