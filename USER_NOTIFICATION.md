## ✨ 新功能

### 1. Excel模板下载 🎉

现在您可以直接下载Excel模板，轻松进行批量导入！

**如何使用**：
1. 进入"设备管理"页面
2. 点击"批量导入"按钮
3. 点击"下载Excel模板"按钮
4. 按照模板格式填写设备信息
5. 上传填写好的文件

**模板特点**：
- ✅ 包含3个示例设备，帮助您快速上手
- ✅ 提供详细的字段说明
- ✅ 自动调整列宽，易于编辑
- ✅ 支持 Excel 2016 及以上版本

## 📋 使用指南

### 批量导入设备（推荐）

1. **下载模板**
   ```
   设备管理 → 批量导入 → 下载Excel模板
   ```

2. **填写设备信息**
   - 参考模板中的示例设备
   - 查看"填写说明"工作表了解字段要求
   - 必填字段：设备ID、品牌、设备名称、规格型号、单价

3. **上传文件**
   ```
   设备管理 → 批量导入 → 选择文件 → 预览数据 → 确认导入
   ```

### 手动添加设备

1. **基本信息**
   - 品牌：从下拉框选择或输入自定义品牌
   - 设备类型：选择后会显示对应的参数表单
   - 设备名称、规格型号：必填

2. **设备参数**
   - 根据设备类型自动显示相关参数字段
   - 填写量程、输出信号等关键参数

3. **详细参数（可选）**
   - 使用自然语言描述特殊参数
   - 例如："立式安装，不锈钢材质，防护等级IP65"
   - 系统会自动提取关键信息用于匹配

## ❓ 常见问题

### Q1：模板下载后无法打开？
**A**：请使用 Microsoft Excel 2016 或更高版本，或者使用 WPS Office、LibreOffice Calc 等兼容软件。

### Q2：品牌下拉框没有我需要的品牌？
**A**：您可以直接输入自定义品牌名称，系统支持自定义品牌。

### Q3：详细参数应该如何填写？
**A**：使用自然语言描述即可，例如：
- ✅ "立式安装，不锈钢材质，防护等级IP65"
- ✅ "工作温度-20~60℃，防护等级IP54"
- ✅ "壁挂式安装，带显示屏"

### Q4：批量导入时出错怎么办？
**A**：
1. 检查Excel文件格式是否正确
2. 确保必填字段都已填写
3. 参考模板中的示例数据格式
4. 查看错误提示信息
