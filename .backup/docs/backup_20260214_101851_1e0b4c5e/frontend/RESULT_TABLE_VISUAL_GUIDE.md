```css
transition: transform 0.2s
hover: transform: translateY(-2px)
```

### 2. 按钮
```css
hover: background-color 变亮
active: 轻微缩放
loading: 旋转动画
```

### 3. 表格行
```css
stripe: 斑马纹交替
hover: 背景色变化
transition: 平滑过渡
```

### 4. 通知
```css
enter: 从右侧滑入
exit: 淡出消失
duration: 3-5秒自动关闭
```

## 可访问性

### 键盘导航
- Tab: 在可交互元素间切换
- Enter: 激活按钮或选择
- Esc: 关闭下拉框
- 方向键: 在下拉选项中导航

### 屏幕阅读器
- 表格使用语义化标签
- 按钮有明确的 aria-label
- 状态变化有文字提示
- 错误信息可被读取

### 颜色对比
- 文字与背景对比度 ≥ 4.5:1
- 重要信息不仅依赖颜色
- 支持高对比度模式
