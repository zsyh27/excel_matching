## 🔧 技术细节

### 设备导入流程

1. **读取Excel文件**: 使用 pandas 读取 Excel 数据
2. **数据提取**: 从说明文本中提取结构化参数
3. **设备类型识别**: 根据类型字符串确定设备类型
4. **生成设备名称**: 基于通径、类型和连接方式生成名称
5. **批量插入**: 使用 SQLAlchemy 批量插入数据库

### 规则生成流程

1. **特征提取**: 使用 `DeviceFeatureExtractor` 提取设备特征
   - 品牌（霍尼韦尔）→ 权重 10.0
   - 设备类型 → 权重 20.0
   - 规格型号 → 权重 5.0

2. **权重分配**: 基于字段类型分配权重
   - device_type字段: 20.0
   - key_params参数: 15.0
   - brand字段: 10.0
   - spec_model字段: 5.0
   - device_name/其他: 1.0

3. **规则保存**: 批量保存到数据库 rules 表
