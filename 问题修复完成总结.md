## 修复的问题

### 1. "水"特征被过滤的问题 ✓ 已解决

**问题描述：**
- 设备详情中显示"介质: 水"，但在匹配规则中找不到"水"这个特征
- 原因：特征质量评分过低（30分），低于最小质量分数要求（50分）

**根本原因：**
- 配置文件`data/static_config.json`中缺少`medium_keywords`字段
- "水"作为单字特征，基础分50分，因为长度<2被扣20分，最终只有30分
- 没有被识别为介质关键词，无法获得+15分的加分

**修复方案：**
在`data/static_config.json`中添加`medium_keywords`字段：

```json
"medium_keywords": [
  "水",
  "气",
  "蒸汽",
  "空气",
  "氮气",
  "油",
  "气体",
  "冷冻水",
  "热水",
  "烟气",
  "粉尘"
]
```

**修复效果：**
- "水"现在被识别为介质关键词
- 质量评分：基础分50 + 在设备关键词中15 = 65分
- 成功通过过滤，可以正常提取为特征

**验证结果：**
```
测试文本: 水
提取的特征: ['水']
质量评分: 65
✓ 配置已生效，'水'特征可以正常提取
```

### 2. 批量导入没有自动生成规则的问题 ✓ 已解决

**问题描述：**
- 批量导入设备时勾选了"自动为导入的设备生成匹配规则"
- 但导入后设备没有生成规则

**根本原因：**
- 后端API使用了错误的方法名：`data_loader.loader.add_rule(rule)` 
- 正确的方法名应该是：`data_loader.loader.save_rule(rule)`

**修复方案：**
修改`backend/app.py`第1950行左右的批量导入代码：

```python
# 修复前
data_loader.loader.add_rule(rule)

# 修复后
data_loader.loader.save_rule(rule)
```

**修复效果：**
- 批量导入时可以正确保存规则到数据库
- 规则生成成功后会在响应中返回生成的规则数量

### 3. 单个设备"生成规则"按钮页面不刷新的问题 ✓ 已解决

**问题描述：**
- 点击设备详情中的"生成规则"按钮
- 提示"规则生成成功"，但页面没有刷新，看不到新生成的规则

**根本原因：**
- 后端API使用了错误的方法名：
  - `rule_gen.generate_rule_for_device(device)` → 应该是 `rule_gen.generate_rule(device)`
  - `data_loader.save_rules()` → 应该是 `data_loader.loader.save_rule(rule)`
- 缺少`preprocessor`参数初始化`RuleGenerator`

**修复方案：**
修改`backend/app.py`第1595行左右的规则重新生成代码：

```python
# 修复前
rule_gen = RuleGenerator(config=config)  # 缺少preprocessor参数
new_rule = rule_gen.generate_rule_for_device(device)  # 错误的方法名
data_loader.save_rules()  # 错误的方法名

# 修复后
rule_gen = RuleGenerator(config=config, preprocessor=preprocessor)  # 添加preprocessor参数
new_rule = rule_gen.generate_rule(device)  # 正确的方法名
data_loader.loader.save_rule(new_rule)  # 正确的方法名
```

**修复效果：**
- 规则生成成功后会正确保存到数据库
- 前端会自动刷新设备详情，显示新生成的规则
- 用户可以立即看到规则的特征列表和权重信息

## 技术细节

### 特征质量评分机制

特征质量评分用于过滤低质量特征，评分规则：

**基础分：** 50分

**加分项：**
- 是技术术语：+20分
- 包含数字：+10分
- 包含单位：+10分
- 在设备关键词库中：+15分（包括brand_keywords、device_type_keywords、medium_keywords）
- 长度适中(3-20)：+5分

**减分项：**
- 是元数据标签：-30分
- 是常见词：-20分
- 太短(<2)：-20分（但如果在设备关键词中则不扣分）
- 纯数字：-5分
- 纯标点：-30分

**最小质量分数：** 50分（可在配置中调整）

### 介质关键词的作用

介质关键词（medium_keywords）用于：
1. 特征质量评分时给予加分（+15分）
2. 识别设备的介质类型
3. 提高介质相关特征的权重

常见介质关键词：
- 水、气、蒸汽、空气、氮气
- 油、气体、冷冻水、热水
- 烟气、粉尘
