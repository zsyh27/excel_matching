3. **液位传感器** - 量程、输出信号、精度

6. **控制器** - 输入点数、输出点数、通讯协议
7. **风机盘管** - 制冷量、制热量、风量
8. **水泵** - 流量、扬程、功率
9. **差压传感器** - 量程、输出信号、精度
10. **电磁阀** - 通径、工作电压、工作压力

### 4. 型号模式扩展

新增型号识别模式：
- `[A-Z]+[0-9]+-[A-Z0-9]+` - 如 VVF53-40
- `[0-9]{3,}[A-Z]{2,}` - 如 2061QAA
- `[A-Z]{2,}_[0-9]+` - 如 QAA_2061

### 5. 参数规则特点

每种设备类型都配置了：
- **关键词列表**：支持中英文、常见别名
- **参数提取规则**：使用正则表达式精确匹配
- **必填/可选标记**：区分核心参数和辅助参数
- **数据类型**：range（范围）、number（数字）、string（字符串）
- **单位信息**：便于数据标准化
