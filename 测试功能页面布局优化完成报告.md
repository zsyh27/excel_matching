## 技术要点

### 1. 页面高度占满

```css
.testing-view {
  min-height: 100vh;  /* 最小高度为视口高度 */
  display: flex;
  flex-direction: column;
  background: #f5f7fa;
}
```

### 2. 内容区域滚动

```css
.content {
  flex: 1;  /* 占据剩余空间 */
  display: flex;
  flex-direction: column;
  gap: 20px;
  padding: 20px;
  overflow-y: auto;  /* 内容超出时显示滚动条 */
}
```

### 3. 模块样式统一

```css
.test-section {
  background: white;
  border-radius: 4px;
  padding: 30px;
  box-shadow: 0 2px 12px 0 rgba(0, 0, 0, 0.1);
}
```
