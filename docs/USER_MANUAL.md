## 系统简介

### 什么是DDC设备清单匹配报价系统？

这是一个智能化的设备报价工具，可以：
- 📊 自动识别Excel中的设备行
- 🤖 智能匹配设备信息
- 💰 自动生成报价清单
- ⚙️ 管理设备库和匹配规则

### 核心优势

- ✅ **高准确率**：设备行识别98%+，匹配准确率91%+
- ✅ **省时高效**：1000行Excel ≤5秒处理
- ✅ **灵活调整**：支持手动调整和批量操作
- ✅ **格式保留**：完整保留原Excel格式

## 快速开始

### 第一次使用

#### 步骤1：访问系统

打开浏览器，访问：`http://localhost:5173`

#### 步骤2：上传Excel文件

1. 点击"上传文件"按钮或拖拽文件到上传区域
2. 支持的格式：.xls, .xlsx, .xlsm
3. 文件大小限制：10MB

#### 步骤3：确认设备行

系统会自动识别设备行，用颜色标记：
- 🟦 **蓝色**：高概率设备行（≥70分）
- 🟨 **黄色**：中概率设备行（40-69分）
- ⬜ **白色**：低概率非设备行（<40分）

如果识别有误，可以手动调整。

#### 步骤4：执行匹配

1. 点击"开始匹配"按钮
2. 系统自动匹配设备信息
3. 查看匹配结果和置信度

#### 步骤5：导出报价单

1. 点击"导出报价单"按钮
2. 下载包含价格的Excel文件
3. 格式与原文件保持一致

## 功能详解

### 1. 文件上传

#### 支持的文件格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| Excel 97-2003 | .xls | 旧版Excel格式 |
| Excel 2007+ | .xlsx | 新版Excel格式 |
| Excel宏文件 | .xlsm | 包含宏的Excel文件 |

#### 上传方式

**方式1：点击上传**
1. 点击"选择文件"按钮
2. 在文件选择器中选择Excel文件
3. 点击"打开"

**方式2：拖拽上传**
1. 打开文件管理器
2. 将Excel文件拖拽到上传区域
3. 松开鼠标完成上传

#### 注意事项

- ⚠️ 文件大小不超过10MB
- ⚠️ 确保Excel文件未损坏
- ⚠️ 建议先备份原文件

### 2. 设备行识别

#### 自动识别

系统使用三维度评分模型自动识别设备行：

1. **内容特征**（40分）
   - 包含设备关键词
   - 包含品牌名称
   - 包含技术参数

2. **格式特征**（30分）
   - 单元格合并情况
   - 字体样式
   - 行高列宽

3. **位置特征**（30分）
   - 在表头之后
   - 连续的数据行
   - 列数匹配

#### 颜色编码

| 颜色 | 分数范围 | 说明 | 建议操作 |
|------|---------|------|---------|
| 🟦 蓝色 | ≥70分 | 高概率设备行 | 通常无需调整 |
| 🟨 黄色 | 40-69分 | 中概率设备行 | 建议人工确认 |
| ⬜ 白色 | <40分 | 低概率非设备行 | 通常为表头或备注 |
| 🟩 绿色 | - | 手动标记为设备行 | 用户确认的设备行 |
| 🟥 红色 | - | 手动取消设备行 | 用户排除的行 |

#### 手动调整

**单行调整**：
1. 找到需要调整的行
2. 点击行右侧的下拉框
3. 选择"标记为设备行"或"取消设备行"

**批量调整**：
1. 勾选多行（点击行首的复选框）
2. 点击顶部的"批量操作"按钮
3. 选择"批量标记"或"批量取消"

#### 筛选功能

使用筛选快速找到需要调整的行：

- **按行号筛选**：输入行号范围（如：10-50）
- **按内容筛选**：输入关键词搜索
- **按概率筛选**：选择高/中/低概率

### 3. 设备匹配

#### 匹配原理

系统使用基于特征权重的匹配算法：

1. **特征提取**：从设备描述中提取关键特征
2. **权重计算**：根据特征类型分配权重
3. **相似度计算**：计算与设备库的相似度
4. **最佳匹配**：选择相似度最高的设备

#### 匹配结果

**成功匹配**：
- 显示匹配的设备信息
- 显示匹配置信度（0-100%）
- 显示设备价格

**匹配失败**：
- 显示"未找到匹配设备"
- 可以手动选择设备
- 或添加新设备到设备库

#### 手动选择设备

如果自动匹配不准确：

1. 点击匹配结果右侧的"更改"按钮
2. 在设备列表中搜索正确的设备
3. 点击"确认"完成选择

### 4. 设备库管理

#### 查看设备库

1. 点击导航栏的"设备管理"
2. 查看所有设备列表
3. 使用搜索和筛选功能

#### 添加设备

**方式1：手动添加**

1. 点击"添加设备"按钮
2. 填写设备信息：
   - 设备ID（自动生成或手动输入）
   - 品牌（从下拉列表选择）
   - 设备名称
   - 规格型号
   - 详细参数
   - 价格
3. 选择设备类型（可选）
4. 填写关键参数（可选）
5. 勾选"自动生成匹配规则"
6. 点击"保存"

**方式2：Excel批量导入**

1. 点击"批量导入"按钮
2. 下载Excel模板
3. 按模板格式填写设备信息
4. 上传填好的Excel文件
5. 查看导入结果

#### 编辑设备

1. 在设备列表中找到要编辑的设备
2. 点击"编辑"按钮
3. 修改设备信息
4. 勾选"重新生成规则"（如果需要）
5. 点击"保存"

#### 删除设备

1. 在设备列表中找到要删除的设备
2. 点击"删除"按钮
3. 确认删除操作
4. 关联的匹配规则会自动删除

#### 动态表单功能

对于支持的设备类型，系统提供动态表单：

**支持的设备类型**：
- 阀门（口径、电压、反馈等）
- 传感器（测量范围、输出信号等）
- 变频器（功率、电压等）
- 执行器（扭矩、行程等）

**使用方法**：
1. 在添加/编辑设备时选择设备类型
2. 系统自动显示该类型的参数字段
3. 填写参数值
4. 系统自动生成结构化的关键参数

### 5. 配置管理

#### 访问配置管理

1. 点击导航栏的"配置管理"
2. 查看所有配置项

#### 主要配置项

**1. 设备类型管理**
- 添加/编辑设备类型
- 配置设备类型关键词
- 设置设备类型参数

**2. 品牌关键词**
- 添加品牌名称
- 配置品牌别名
- 设置品牌识别规则

**3. 归一化规则**
- 配置文本归一化映射
- 统一单位表示
- 标准化符号

**4. 同义词映射**
- 配置同义词关系
- 提高匹配召回率

**5. 特征权重**
- 调整特征权重
- 优化匹配准确率

**6. 设备行识别**
- 调整识别阈值
- 配置识别规则

#### 修改配置

1. 找到要修改的配置项
2. 点击"编辑"按钮
3. 修改配置值
4. 点击"保存"
5. 系统自动重新初始化相关组件

⚠️ **注意**：修改配置会影响匹配结果，建议谨慎操作。

### 6. 统计仪表板

#### 访问统计仪表板

1. 点击导航栏的"统计"
2. 查看数据统计信息

#### 统计内容

**概览卡片**：
- 设备总数
- 规则总数
- 品牌数量
- 规则覆盖率

**品牌分布图**：
- 各品牌设备数量
- 点击柱状图可筛选

**价格分布图**：
- 各价格区间设备数量
- 点击区间可筛选

**最近添加**：
- 最新10条设备记录
- 点击可查看详情

**无规则设备**：
- 没有匹配规则的设备列表
- 可批量生成规则

### 7. 数据一致性检查

#### 执行检查

1. 在设备管理页面点击"一致性检查"
2. 系统自动检查数据一致性
3. 查看检查报告

#### 检查内容

- 没有规则的设备
- 孤立规则（关联设备不存在）
- 数据完整性问题

#### 修复问题

1. 查看检查报告
2. 勾选修复选项：
   - 为缺失规则的设备生成规则
   - 删除孤立规则
3. 点击"执行修复"
4. 查看修复结果

## 常见问题

### Q1: 为什么有些设备行没有被识别？

**可能原因**：
- 设备描述不包含关键词
- 行格式与其他设备行差异较大
- 位置不在预期范围内

**解决方法**：
- 手动标记为设备行
- 调整设备行识别配置
- 优化Excel文件格式

### Q2: 为什么匹配失败？

**可能原因**：
- 设备库中没有该设备
- 设备描述信息不足
- 匹配规则不准确

**解决方法**：
- 添加设备到设备库
- 补充设备描述信息
- 手动选择正确的设备
- 优化匹配规则

### Q3: 如何提高匹配准确率？

**建议**：
1. 完善设备库信息
2. 为所有设备生成规则
3. 配置同义词映射
4. 调整特征权重
5. 使用动态表单录入设备

### Q4: 导出的Excel格式不对？

**检查**：
- 原Excel文件是否正常
- 是否使用了不支持的格式
- 文件是否过大

**解决**：
- 使用标准Excel格式
- 简化Excel文件结构
- 分批处理大文件

### Q5: 如何备份数据？

**数据库模式**：
```bash
# 备份SQLite数据库
copy data\devices.db data\devices_backup.db

# 备份MySQL数据库
mysqldump -u user -p ddc_devices > backup.sql
```

**JSON模式**：
```bash
# 备份JSON文件
copy data\static_device.json data\static_device_backup.json
copy data\static_rule.json data\static_rule_backup.json
```

### Q6: 如何恢复数据？

**数据库模式**：
```bash
# 恢复SQLite数据库
copy data\devices_backup.db data\devices.db

# 恢复MySQL数据库
mysql -u user -p ddc_devices < backup.sql
```

**JSON模式**：
```bash
# 恢复JSON文件
copy data\static_device_backup.json data\static_device.json
copy data\static_rule_backup.json data\static_rule.json
```

## 最佳实践

### 1. Excel文件准备

**建议格式**：
- 使用标准的表格结构
- 设备信息在连续的行中
- 避免过多的合并单元格
- 使用清晰的表头

**示例**：

| 序号 | 设备名称 | 品牌 | 规格型号 | 详细参数 | 数量 | 单价 |
|------|---------|------|---------|---------|------|------|
| 1 | 电动二通阀 | 霍尼韦尔 | VC4013 | DN15 AC24V | 10 | |
| 2 | 温度传感器 | 西门子 | QAA2012 | 0-50℃ | 5 | |

### 2. 设备库维护

**定期维护**：
- 每月检查数据一致性
- 及时添加新设备
- 更新设备价格
- 优化匹配规则

**命名规范**：
- 设备ID：使用统一格式（如DEV001）
- 设备名称：使用标准名称
- 品牌：使用官方名称

### 3. 匹配优化

**提高准确率**：
1. 完善设备详细参数
2. 使用动态表单录入
3. 配置同义词映射
4. 定期重新生成规则

**处理特殊情况**：
- 对于相似设备，补充区分特征
- 对于新型号设备，及时添加到设备库
- 对于匹配失败的设备，手动选择并记录

### 4. 性能优化

**大文件处理**：
- 分批上传（每批≤1000行）
- 使用分页查询
- 定期清理无用数据

**系统维护**：
- 定期备份数据
- 清理临时文件
- 优化数据库索引

### 5. 团队协作

**权限管理**（未来版本）：
- 设置不同用户角色
- 控制操作权限
- 记录操作日志

**数据共享**：
- 导出设备库供团队使用
- 共享配置文件
- 统一匹配规则
