## 🚀 使用指南

### 1. 访问测试功能页面

**方法1**: 点击主菜单"测试功能"
- 位置：主页面右上角菜单栏
- 效果：在新标签页打开测试功能页面

**方法2**: 直接访问URL
```
http://localhost:3000/testing
```

### 2. 使用设备类型识别测试

1. 点击左侧"设备类型识别测试"
2. 在输入框中输入设备描述
   - 例如：`CO浓度探测器`
   - 例如：`温度传感器`
   - 例如：`蝶阀`
3. 等待识别结果（自动触发，500ms防抖）
4. 查看识别结果：
   - 设备类型
   - 置信度
   - 识别模式
   - 关键词

### 3. 使用六步流程实时预览

1. 点击左侧"六步流程实时预览"
2. 在输入框中输入设备描述
   - 例如：`CO浓度探测器 量程0~250ppm 输出4~20mA 精度±5%`
3. 等待分析结果（自动触发，500ms防抖）
4. 查看六步流程：
   - 步骤0：文本预处理
   - 步骤1：设备类型识别
   - 步骤2：技术参数提取
   - 步骤3：辅助信息提取
   - 步骤4：智能匹配评分
   - 步骤5：用户界面展示
   - 性能统计
