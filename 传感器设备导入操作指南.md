# 传感器设备导入操作指南

## 概述

本指南说明如何配置设备类型参数并导入传感器设备数据。

## 设备类型和参数

### 需要配置的设备类型

1. **温度传感器** (22个设备)
2. **温湿度传感器** (80个设备)
3. **空气质量传感器** (35个设备)

### 所有设备类型的通用参数列表

以下16个参数适用于所有设备类型，用户可根据实际情况选择填写：

1. 检测对象
2. 安装位置
3. 温度信号类型
4. 温度量程
5. 温度精度
6. 湿度信号类型
7. 湿度量程
8. 湿度精度
9. 浓度量程
10. 浓度精度
11. 输出信号
12. 通道数
13. 显示屏
14. 继电器
15. 面板颜色
16. 电源

## 操作步骤

### 步骤1：配置设备类型参数

#### 方式一：通过前端界面配置（推荐）

1. 访问配置管理页面：http://localhost:3000/config-management

2. 在左侧菜单选择：**设备参数配置 > 设备类型**

3. 为每个设备类型添加参数：

   **温度传感器：**
   - 点击"添加设备类型"
   - 设备类型名称：`温度传感器`
   - 添加以下参数（全部设为可选）：
     - 检测对象 (text, 可选)
     - 安装位置 (text, 可选)
     - 温度信号类型 (text, 可选)
     - 温度量程 (text, 可选)
     - 温度精度 (text, 可选)
     - 湿度信号类型 (text, 可选)
     - 湿度量程 (text, 可选)
     - 湿度精度 (text, 可选)
     - 浓度量程 (text, 可选)
     - 浓度精度 (text, 可选)
     - 输出信号 (text, 可选)
     - 通道数 (text, 可选)
     - 显示屏 (text, 可选)
     - 继电器 (text, 可选)
     - 面板颜色 (text, 可选)
     - 电源 (text, 可选)

   **温湿度传感器：**
   - 重复上述步骤，设备类型名称改为：`温湿度传感器`
   - 添加相同的16个参数

   **空气质量传感器：**
   - 重复上述步骤，设备类型名称改为：`空气质量传感器`
   - 添加相同的16个参数

4. 保存配置

#### 方式二：通过API配置（快速）

运行以下Python脚本：

```bash
python import_sensor_devices.py
```

该脚本会自动：
- 配置3个设备类型及其参数
- 导入Excel中的137个设备

### 步骤2：导入Excel数据

#### 方式一：通过前端界面导入

1. 访问设备管理页面：http://localhost:3000/database/devices

2. 点击"批量导入"按钮

3. 选择文件：`data/传感器设备_标准化.xlsx`

4. 点击上传

5. 等待导入完成，查看导入结果

#### 方式二：通过脚本导入

如果已经运行了 `import_sensor_devices.py`，数据会自动导入。

## 参数管理功能

### 参数排序

在设备参数配置界面，每个参数都有上移/下移按钮：

- **上移** ↑：将参数向上移动一位
- **下移** ↓：将参数向下移动一位

这样可以调整参数在设备录入表单中的显示顺序。

### 参数删除

每个参数旁边有删除按钮 🗑️：

- 点击删除按钮可以移除不需要的参数
- 删除后该参数不会在设备录入表单中显示

### 参数添加

在参数列表底部有"添加参数"按钮：

- 点击可以添加新的参数
- 填写参数名称、类型（text/number/select）
- 设置是否必填

## 注意事项

1. **价格格式**：Excel中的价格如果是浮点数，会自动转换为整数（四舍五入）

2. **参数可选**：所有参数都是可选的，没有数据的列可以留空

3. **设备类型必须匹配**：Excel中的"设备类型"必须与配置中的设备类型名称完全一致

4. **编码问题**：确保Excel文件使用UTF-8编码，避免中文乱码

5. **数据验证**：导入前系统会验证必需字段，缺少必需字段的行会被跳过

## 常见问题

### Q1: 导入失败，提示"设备类型不存在"

**解决方案**：先在配置管理中添加对应的设备类型配置

### Q2: 参数没有显示在设备详情中

**解决方案**：检查设备类型配置中是否添加了该参数

### Q3: 价格显示不正确

**解决方案**：价格已改为整数类型，小数会被四舍五入

### Q4: 如何调整参数显示顺序？

**解决方案**：在设备参数配置界面使用上移/下移按钮调整顺序

### Q5: 如何删除不需要的参数？

**解决方案**：在设备参数配置界面点击参数旁边的删除按钮
