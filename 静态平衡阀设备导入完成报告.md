## 关键技术点

### 1. Excel字段映射

原始Excel文件使用不同的字段名，需要进行映射：

```python
field_mapping = {
    '型号': 'spec_model',
    '价格': 'unit_price',
    '说明': 'description',
    '设备类型': 'device_type'
}
```

### 2. 参数解析

从"说明"字段解析结构化参数：

```python
# 格式：参数名：参数值，参数名：参数值
params = description.split('，')
for param in params:
    if '：' in param:
        key, value = param.split('：', 1)
        key_params[key.strip()] = {"value": value.strip()}
```

### 3. 设备ID生成

使用设备类型前缀和型号生成唯一ID：

```python
prefix = device_type_prefixes.get(device_type, 'DEV')
device_id = f"HON_{prefix}_{spec_model.replace('-', '_').replace('/', '_')}"
```

### 4. 特征提取验证

确保特征数量正确：

```python
expected_feature_count = 4 + len(device.key_params)
# 4个基础特征：品牌、设备类型、设备名称、规格型号
# + key_params中的所有参数
```
