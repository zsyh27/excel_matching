# 设备表单修复快速测试指南

1. 打开浏览器访问 `http://localhost:5173`

- 确认 `static_config.json` 中 `brand_keywords` 是数组格式

## 常见问题排查

### 问题1：品牌下拉框仍显示数字

**可能原因**：
- 前端代码未更新
- 浏览器缓存未清除

**解决方法**：
```bash
# 1. 确认代码已更新
git status

# 2. 清除浏览器缓存
# Chrome: Ctrl+Shift+Delete

# 3. 重启前端服务器
cd frontend
npm run dev
```

### 问题2：下载模板按钮无响应

**可能原因**：
- xlsx 库未安装
- 浏览器阻止下载

**解决方法**：
```bash
# 1. 检查 xlsx 是否安装
cd frontend
npm list xlsx

# 2. 重新安装依赖
npm install

# 3. 检查浏览器控制台错误
# 打开开发者工具 (F12) 查看错误信息
```

### 问题3：模板下载后无法打开

**可能原因**：
- Excel 版本不兼容
- 文件损坏

**解决方法**：
- 使用 Microsoft Excel 2016 或更高版本
- 使用 WPS Office
- 使用 LibreOffice Calc
