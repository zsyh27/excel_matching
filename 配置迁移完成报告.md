### 步骤2: 修改backend/app.py ✅

**修改前**:
```python
# 临时加载配置用于初始化预处理器
from modules.data_loader import ConfigManager
temp_config_manager = ConfigManager(Config.CONFIG_FILE)
config = temp_config_manager.get_config()
preprocessor = TextPreprocessor(config)

# 使用新方式初始化数据加载器
data_loader = DataLoader(config=Config, preprocessor=preprocessor)

# 重新加载配置
config = data_loader.load_config()
```

**修改后**:
```python
# 初始化数据加载器（数据库模式）
# 注意：配置已迁移到数据库，不再使用JSON文件
data_loader = DataLoader(config=Config, preprocessor=None)

# 从数据库加载配置
config = data_loader.load_config()

# 使用配置初始化文本预处理器
preprocessor = TextPreprocessor(config)
data_loader.preprocessor = preprocessor
```

**改进**:
- ❌ 移除了对JSON文件的依赖
- ✅ 直接从数据库加载配置
- ✅ 简化了初始化流程
