## 一、匹配阶段完整流程

### 1.1 五步处理流程（智能提取系统）

根据 `.kiro/steering/intelligent-extraction-system-guide.md`，智能提取和匹配系统采用五步流程：

```
步骤1: 设备类型识别 (准确率 >85%)
   ↓
步骤2: 技术参数提取 (量程、输出、精度、规格)
   ↓
步骤3: 辅助信息提取 (品牌、介质、型号)
   ↓
步骤4: 智能匹配和评分 (多维度评分)
   ↓
步骤5: 用户界面展示 (按评分排序，默认选中最高分)
```

### 1.2 用户操作流程

```
1. 用户上传Excel文件
   ↓
2. 系统识别设备行（设备行识别配置）
   ↓
3. 对每个设备行执行五步流程
   ↓
4. 前端展示匹配结果（下拉框，默认选中最高分设备）
   ↓
5. 用户可以：
   - 查看候选设备列表（前20个）
   - 手动选择其他候选设备
   - 查看匹配详情（评分明细）
   - 筛选设备类型
   - 导出匹配结果
```

### 1.3 后端API接口

| 接口 | 方法 | 功能 | 状态 |
|------|------|------|------|
| `/api/match` | POST | 批量匹配设备 | ✅ 已实现 |
| `/api/match/detail/<cache_key>` | GET | 获取匹配详情 | ✅ 已实现 |
| `/api/match/detail/export/<cache_key>` | GET | 导出匹配详情 | ✅ 已实现 |
| `/api/intelligent-extraction/extract` | POST | 提取设备信息 | ✅ 已实现（代码中） |
| `/api/intelligent-extraction/match` | POST | 智能匹配单个设备 | ✅ 已实现（代码中） |
| `/api/intelligent-extraction/match-batch` | POST | 批量智能匹配 | ✅ 已实现（代码中） |
| `/api/intelligent-extraction/preview` | POST | 五步流程预览 | ✅ 已实现（代码中） |

**注意**: `intelligent-extraction` 相关接口在 `backend/modules/intelligent_extraction/api_handler.py` 中已实现，但需要确认是否在 `backend/app.py` 中注册了路由。

### 4.2 API集成

在 `frontend/src/api/` 目录下创建 `intelligentExtraction.js`：

```javascript
import request from '@/utils/request'

/**
 * 提取设备信息
 */
export function extractDeviceInfo(text) {
  return request({
    url: '/api/intelligent-extraction/extract',
    method: 'post',
    data: { text }
  })
}

/**
 * 智能匹配设备
 */
export function matchDevice(text, topK = 5) {
  return request({
    url: '/api/intelligent-extraction/match',
    method: 'post',
    data: { text, top_k: topK }
  })
}

/**
 * 五步流程预览
 */
export function previewExtraction(text) {
  return request({
    url: '/api/intelligent-extraction/preview',
    method: 'post',
    data: { text }
  })
}
```

### 4.3 后端路由注册

需要在 `backend/app.py` 中添加智能提取API路由（如果尚未添加）：

```python
from modules.intelligent_extraction.api_handler import IntelligentExtractionAPI

# 初始化智能提取API
intelligent_extraction_api = IntelligentExtractionAPI(config, device_loader)

@app.route('/api/intelligent-extraction/extract', methods=['POST'])
def intelligent_extract():
    """提取设备信息"""
    data = request.get_json()
    text = data.get('text', '')
    result = intelligent_extraction_api.extract(text)
    return jsonify(result)

@app.route('/api/intelligent-extraction/match', methods=['POST'])
def intelligent_match():
    """智能匹配设备"""
    data = request.get_json()
    text = data.get('text', '')
    top_k = data.get('top_k', 5)
    result = intelligent_extraction_api.match(text, top_k)
    return jsonify(result)

@app.route('/api/intelligent-extraction/preview', methods=['POST'])
def intelligent_preview():
    """五步流程预览"""
    data = request.get_json()
    text = data.get('text', '')
    result = intelligent_extraction_api.preview(text)
    return jsonify(result)
```
