## 常见问题

### Q1: 为什么有些设备的"霍尼韦尔"权重是15，有些是10？

**A:** 这取决于特征的来源：
- 如果来自 `brand` 字段：权重为 10.0
- 如果来自 `key_params`（如"执行器品牌"）：权重为 15.0

这是正确的行为，因为 key_params 中的参数被认为是更重要的识别特征。

### Q2: 批量导入后看不到规则怎么办？

**A:** 请按以下步骤检查：
1. 刷新页面（F5）
2. 运行检查脚本：`python backend/check_batch_import_issue.py`
3. 查看后端日志，确认规则生成是否成功
4. 如果规则确实没有生成，检查是否勾选了"自动生成规则"选项

### Q3: 关键参数显示为空怎么办？

**A:** 
1. 确认已执行步骤2（重启后端服务）
2. 确认已刷新前端页面
3. 检查浏览器控制台是否有错误
4. 运行检查脚本确认数据库中有key_params数据

### Q4: 如何验证权重是否正确？

**A:** 运行测试脚本：
```bash
cd backend
python test_spec_model_weight.py
```

查看输出，确认所有权重都正确。
