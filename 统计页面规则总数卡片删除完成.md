# 统计页面规则总数卡片删除完成

## 修改内容

删除了统计页面系统概览 tab 中的"规则总数"和"规则覆盖率"卡片。

### 修改前（4个卡片）

```
┌─────────────┬─────────────┬─────────────┬─────────────┐
│  设备总数   │  规则总数   │  品牌数量   │ 规则覆盖率  │
│   (span=6)  │   (span=6)  │   (span=6)  │   (span=6)  │
└─────────────┴─────────────┴─────────────┴─────────────┘
```

### 修改后（3个卡片）

```
┌──────────────────┬──────────────────┬──────────────────┐
│    设备总数      │    品牌数量      │    设备类型      │
│    (span=8)      │    (span=8)      │    (span=8)      │
└──────────────────┴──────────────────┴──────────────────┘
```

## 具体变化

### 删除的卡片

1. **规则总数** - 显示 `statistics.total_rules`
2. **规则覆盖率** - 显示规则数/设备数的百分比

### 保留的卡片

1. **设备总数** - 显示 `statistics.total_devices`
2. **品牌数量** - 显示 `statistics.total_brands`
3. **设备类型** - 显示 `statistics.total_device_types`（新增）

### 布局调整

- 卡片宽度从 `span="6"` 改为 `span="8"`
- 3个卡片平均分配24列宽度（每个8列）

### 代码清理

- 移除了 `Document` 和 `PieChart` 图标导入
- 移除了 `getCoverageRate()` 和 `getCoverageColor()` 计算函数
- 移除了 `computed` 的导入（不再需要）
- 添加了 `Histogram` 图标用于设备类型卡片

## 原因

根据系统架构变更，旧的规则系统已被智能提取匹配系统替代，因此规则总数和规则覆盖率指标不再适用。
