## 如何访问配置

### 方法1：通过 DatabaseLoader 访问（推荐）

```python
from modules.database import DatabaseManager
from modules.database_loader import DatabaseLoader

# 初始化
db_manager = DatabaseManager("sqlite:///data/devices.db")
db_loader = DatabaseLoader(db_manager)

# 访问设备参数配置（原 device_params.yaml）
device_params = db_loader.get_config_by_key('device_params')
print(device_params['brands'])  # 品牌配置
print(device_params['device_types'])  # 设备类型配置

# 访问智能提取配置
intelligent_extraction = db_loader.get_config_by_key('intelligent_extraction')

# 访问其他配置
brand_keywords = db_loader.get_config_by_key('brand_keywords')
device_type_keywords = db_loader.get_config_by_key('device_type_keywords')

# 加载所有配置
all_configs = db_loader.load_config()
```

### 方法2：通过 static_config.json 访问

```python
import json

# 读取主配置文件
with open('data/static_config.json', 'r', encoding='utf-8') as f:
    config = json.load(f)

# 访问配置
device_type_keywords = config.get('device_type_keywords', {})
brand_keywords = config.get('brand_keywords', {})
```
