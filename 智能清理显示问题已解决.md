### 用户体验改进

**之前（智能清理启用时）：**
```
步骤0：文本预处理
  原始文本: 1.名称:CO浓度探测器 2.规格参数：...3.施工要求:...
  清理后文本: CO浓度探测器+电化学式++量程+0~250ppm+...
  归一化文本: co浓度探测器+电化学式++量程+0~250ppm+...
  智能清理: 应用规则: truncation, metadata_tag, separator_unification
           删除长度: 57 字符
```

**现在（智能清理禁用后）：**
```
步骤0：文本预处理
  原始文本: CO浓度探测器 量程0~250ppm 输出4~20mA 精度±5%
  归一化文本: co浓度探测器量程0~250ppm输出4~20ma精度±5%
  提取特征: 5 个
  归一化映射: 0 个映射
  全局配置: fullwidth_to_halfwidth, unify_lowercase
  特征提取: 提取: 5 个, 过滤: 0 个
```

### 关键改进

1. **简化显示**：移除了混淆的"清理后文本"和"智能清理"信息
2. **保持一致**：现在的处理流程更清晰：原始文本 → 归一化 → 特征提取
3. **符合新系统**：与新的智能提取系统（直接提取）保持一致
