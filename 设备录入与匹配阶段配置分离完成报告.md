# 设备录入与匹配阶段配置分离完成报告

## 问题背景

用户发现"温度"、"压力"等词存在于元数据关键词配置中,是因为后期需要匹配的设备有混乱的输入,为了匹配成功而增加的。但这些配置也影响了设备录入阶段,导致设备类型被错误拆分。

### 核心矛盾

- **元数据关键词和白名单**是为**匹配阶段**设计的,用于处理混乱的Excel输入
- 但这些配置也被应用到了**设备录入阶段**,导致设备类型被错误拆分

## 解决方案

实施**方案1: 在设备录入阶段跳过元数据关键词处理**

### 核心思想

通过`mode`参数区分两个阶段:
- `mode='device'` - 设备录入阶段,跳过元数据关键词处理
- `mode='matching'` - 匹配阶段,应用元数据关键词处理

## 代码修改

### 修改文件

`backend/modules/text_preprocessor.py`

### 修改内容

#### 1. 修改`_extract_bracket_features`方法签名

```python
# 修改前
def _extract_bracket_features(self, text: str) -> List[str]:

# 修改后
def _extract_bracket_features(self, text: str, mode: str = 'matching') -> List[str]:
```

#### 2. 添加mode判断逻辑

```python
# 只在匹配模式下删除元数据关键词前缀
# 在设备录入模式下跳过,避免设备类型被拆分
if mode == 'matching':
    # 首先检查并删除元数据关键词前缀
    sorted_metadata_keywords = sorted(self.metadata_keywords, key=len, reverse=True)
    for keyword in sorted_metadata_keywords:
        keyword_lower = keyword.lower()
        text_lower = text.lower()
        if text_lower.startswith(keyword_lower):
            text = text[len(keyword):]
            break
```

#### 3. 更新调用点

```python
# 在_extract_features_with_detail方法中
# 修改前
bracket_features = self._extract_bracket_features(segment)

# 修改后
bracket_features = self._extract_bracket_features(segment, mode=mode)
```

## 配置管理建议

### 现在可以做的

用户现在可以放心地在"配置管理 → 高级配置 → 元数据关键词"中添加任何需要的关键词,包括:

- ✅ 温度
- ✅ 压力
- ✅ 湿度
- ✅ 流量
- ✅ 功率

这些关键词只会在匹配阶段生效,不会影响设备录入阶段的规则生成。

### 配置原则

1. **元数据关键词**: 添加所有可能出现在Excel混乱输入中的字段名称
2. **特征白名单**: 添加重要但简短的特征(如"水"、"气"、"阀")
3. **不用担心**: 这些配置不会影响设备库的规则生成

## 技术细节

### mode参数的传递路径

```
preprocess(text, mode='device')
  ↓
_extract_features_with_detail(text, mode='device')
  ↓
_extract_bracket_features(segment, mode='device')
  ↓
if mode == 'matching':  # 跳过元数据关键词处理
    ...
```

### 使用mode='device'的地方

1. `backend/modules/rule_generator.py`
   - `extract_features` 方法
   - 所有设备规则生成都使用`mode='device'`

2. `backend/modules/text_preprocessor.py`
   - `_normalize_with_detail` 方法
   - 在设备模式下跳过温度单位的映射

### 使用mode='matching'的地方

1. Excel匹配引擎
2. 批量导入预览
3. 所有需要处理用户输入的地方
