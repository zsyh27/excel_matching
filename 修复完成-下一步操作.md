## ❓ 常见问题

### Q: 重启后端后仍然显示旧格式？

**A**: 检查以下几点：
1. 确认所有 `__pycache__` 文件夹都已删除
2. 确认后端服务已完全重启（不是热重载）
3. 浏览器强制刷新（Ctrl+Shift+R）
4. 检查浏览器控制台是否有错误

### Q: 如何确认后端代码已更新？

**A**: 运行测试脚本：
```bash
python test_frontend_display.py
```
如果测试通过，说明后端代码正确。

### Q: 浏览器显示不正确但测试脚本通过？

**A**: 说明是前端问题：
1. 确认 `FiveStepPreview.vue` 文件已更新
2. 重启前端开发服务器（如果使用）
3. 清除浏览器缓存
4. 强制刷新页面

### Q: 如何查看详细的错误信息？

**A**: 
1. 打开浏览器开发者工具（F12）
2. 切换到 Console 标签查看JavaScript错误
3. 切换到 Network 标签查看API请求和响应
4. 查看后端控制台日志
