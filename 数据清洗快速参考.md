# 数据清洗快速参考卡片

## 🎯 下次这样告诉我

```
我有【数量】条【品牌】【设备类型】数据需要清洗并生成Excel。

数据格式：【简单说明，如：型号 价格 参数描述】

数据内容：
【直接粘贴原始数据】
```

---

## 📋 推荐的数据格式

### 格式1：空格分隔（最简单）⭐

```
型号 价格 参数描述
P7620C0042A 4516 0...4 Bar, H-Port<20 Bar, 4-20mA
P7620C0042B 4516 0...4 Bar, L-Port<20 Bar, 0-10V
```

### 格式2：结构化文本（最清晰）⭐⭐

```
型号: P7620C0042A | 价格: 4516 | 量程: 0-4 Bar | 输出: 4-20mA
型号: P7620C0042B | 价格: 4516 | 量程: 0-4 Bar | 输出: 0-10V
```

### 格式3：CSV（大量数据）

```csv
型号,价格,量程,输出信号
P7620C0042A,4516,0-4 Bar,4-20mA
P7620C0042B,4516,0-4 Bar,0-10V
```

---

## ✅ 最佳实践

1. **一次性提供全部数据**（不要分批）
2. **说明字段含义**（简单说明即可）
3. **指定品牌名称**（如：霍尼韦尔、西门子）
4. **保持格式统一**（同一批数据用同一格式）

---

## 📊 生成的Excel格式

| 品牌 | 设备类型 | 设备名称 | 规格型号 | 单价 | 量程 | 输出信号 | 精度 | ... |
|------|---------|---------|---------|------|------|---------|------|-----|
| 霍尼韦尔 | 压差变送器 | 陶瓷芯体压差变送器 | P7620C0042A | 4516 | 0-4 Bar | 4-20mA | TC-0<0.08% | ... |

**说明**：
- 前5列是标准字段（必填）
- 后面的列是参数列（自动转为key_params，权重15.0）

---

## 🚀 完整示例

```
我有120条霍尼韦尔暖通设备数据需要清洗并生成Excel。

数据格式：序号 型号 价格1 价格2 参数描述

数据内容：
82 P7620C0042A 4516 4516 0...4 Bar, H-Port<20 Bar, L-Port<20 Bar, TC-0<0.08%, 4-20mA
83 P7620C0042B 4516 4516 0...4 Bar, H-Port<20 Bar, L-Port<20 Bar, TC-0<0.08%, 0-10V
84 P7620C0060A 5067 5067 0…6 Bar, H-Port<12 Bar, L-Port<12 Bar, TC-0<0.04%, 4-20mA
...（共120条）
```
