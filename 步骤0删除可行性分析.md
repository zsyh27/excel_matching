# 步骤0删除可行性分析

## 你的问题

1. **既然步骤0不参与后续的步骤1-5的工作，仅仅用于展示，那就可以直接删除步骤0，是不是？**
2. **步骤1的工作是提取文本中的设备类型关键字，步骤2-3的工作是提取文本中的技术参数和辅助信息关键字，提取的目的是为了在设备表中进行匹配对不对？**

## 核心回答

### 问题1：可以删除步骤0吗？

**答案：不能完全删除，但可以简化或隐藏！**

#### ❌ 不能完全删除的原因

通过代码分析，发现步骤0（文本预处理）在系统中有**两个重要用途**：

##### 用途1：旧的规则匹配引擎（MatchEngine）仍在使用 ⚠️

## 评分算法详解

### 总分计算公式

```
总分 = 设备类型得分 × 50% + 参数得分 × 30% + 品牌得分 × 10% + 其他得分 × 10%
```

### 各项得分计算

#### 1. 设备类型得分（50%权重）

```python
def _score_device_type(self, extraction, device):
    # 完全匹配：1.0（50分）
    if extracted_type == device_type:
        return 1.0
    
    # 主类型匹配 + 2个关键词：0.9（45分）
    if extracted_main in device_type and keyword_match >= 2:
        return 0.9
    
    # 主类型匹配 + 1个关键词：0.8（40分）
    if extracted_main in device_type and keyword_match == 1:
        return 0.8
    
    # 主类型匹配：0.7（35分）
    if extracted_main in device_type:
        return 0.7
    
    # 相近类型：0.5（25分）
    return 0.5
```

#### 2. 参数得分（30%权重）

```python
def _score_parameters(self, extraction, device):
    score = 0.0
    
    # 量程匹配：0.5（15分）
    if ranges_exact_match:
        score += 0.5
    elif ranges_overlap:
        score += 0.33
    
    # 输出信号匹配：0.33（10分）
    if outputs_match:
        score += 0.33
    elif outputs_equivalent:
        score += 0.23
    
    # 精度匹配：0.17（5分）
    if accuracy_match:
        score += 0.17
    
    return score  # 最高1.0（30分）
```

#### 3. 品牌得分（10%权重）

```python
def _score_brand(self, extraction, device):
    # 品牌一致：1.0（10分）
    if extraction.brand == device.brand:
        return 1.0
    
    # 品牌不一致：0.0（0分）
    return 0.0
```

#### 4. 其他得分（10%权重）

```python
def _score_others(self, extraction, device):
    score = 0.0
    
    # 介质匹配：0.5（5分）
    if medium_match:
        score += 0.5
    
    # 型号匹配：0.5（5分）
    if model_match:
        score += 0.5
    
    return score  # 最高1.0（10分）
```

## 多阶段匹配策略

### 第一阶段：严格匹配（90+分）

- 设备类型完全匹配
- 主要参数都匹配
- 品牌一致

### 第二阶段：宽松匹配（70-89分）

- 设备类型匹配
- 参数部分匹配
- 品牌可能不一致

### 第三阶段：模糊匹配（50-69分）

- 主类型匹配
- 参数模糊匹配
- 允许更大的容差

### 第四阶段：兜底匹配（30-49分）

- 返回相近类型的设备
- 作为最后的候选
