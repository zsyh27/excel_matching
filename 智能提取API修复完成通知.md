## 注意事项

### 匹配结果为空的情况

如果测试时返回空列表（没有匹配的设备），这是正常的，可能的原因：

1. **数据库中没有该类型的设备**
   - 解决方案：先导入相应类型的设备数据

2. **匹配阈值过高**
   - 当前阈值：兜底匹配 30分
   - 如果设备得分低于30分，不会返回

3. **设备类型配置不完整**
   - 确保在"设备类型模式"中正确配置了设备类型和前缀关键词

### 性能说明

- 单次测试耗时：约 600-800ms（包含完整的五步流程）
- 数据库查询：每个匹配阶段会查询一次数据库
- 设备数量：当前数据库有 3051 个设备
