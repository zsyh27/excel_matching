## 三、用户操作指南

### 步骤1：刷新配置管理页面
1. 打开浏览器中的配置管理页面
2. 按 `F5` 或 `Ctrl+R` 刷新页面
3. 如果仍未显示，请重启后端服务：
   ```bash
   # 停止当前后端服务（Ctrl+C）
   # 重新启动
   cd backend
   python app.py
   ```

### 步骤2：验证配置是否显示

#### 2.1 检查设备参数配置
1. 进入"配置管理" → "设备参数配置"
2. 应该能看到15种设备类型，包括新增的5种传感器：
   - ✅ 温度传感器
   - ✅ 温湿度传感器
   - ✅ CO2传感器
   - ✅ CO传感器
   - ✅ 空气质量传感器
3. 点击任一传感器类型，查看参数配置是否正确

#### 2.2 检查特征白名单
1. 进入"配置管理" → "特征白名单"
2. 搜索传感器相关特征：
   - ✅ `co`
   - ✅ `co2`
   - ✅ `pm2.5`
   - ✅ `pm10`
   - ✅ `温度`
   - ✅ `湿度`
   - ✅ `霍尼韦尔`

#### 2.3 检查同义词映射
1. 进入"配置管理" → "同义词映射"
2. 应该能看到正确的键值对，例如：
   - ✅ `co` → `一氧化碳`
   - ✅ `co2` → `二氧化碳`
   - ✅ `温度传感器` → `温度探头, 温度探测器, 温度检测器`
3. **不应该**看到错误的格式如：
   - ❌ `synonym_map` → `json`

### 步骤3：导入传感器设备
1. 进入"设备管理"页面
2. 点击"批量导入"按钮
3. 选择文件：`data/传感器设备_标准化.xlsx`
4. 上传并等待导入完成
5. 系统会自动：
   - 识别设备类型
   - 提取设备参数
   - 生成匹配规则
